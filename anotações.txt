Para converter uma resposta de inteiro para string, basta utilizar String() e dentro do valor string colocar a variável que irá devolver o valor em inteiro

Exemplo:

    var n1 = Number.parseFloat(window.prompt('Digite um número: '));                    OU                  var n1 = Number(window.prompt('Digite um número: '));
    var n2 = Number.parseFloat(window.prompt('Digite outro número: '));                    OU                  var n1 = Number(window.prompt('Digite outro número: '));            

    var soma = n1 + n2;
    window.prompt('A soma dos valores é' + String(soma));


parseFloat - 5.5
parseInt - 5


Formatando Strings
    
soma.length() - Quantos caracteres a string tem
soma.toUpperCase() - Tudo para letras 'MAIÚSCULAS'
soma.toLowerCase() - Tudo para letras 'minúsculas'

var n1 = 1545.5

n1.toFixed(2) - Representa o número de casas decimais
n1.toFixed(2).replace('.', ',') - Para colocar a ordem do potno e da vírgula no valor salário

n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) - Código para você retornar o valor em um saldo monetário
